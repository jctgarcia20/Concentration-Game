<h1>Description</h1>
<p>Test your memory by matching the two same cards!
<br>
<h2>HOW TO PLAY:</h2>
<p> - Match the same pair of identical cards (64 cards total, 32 pairs)
<br> - Try to match all the cards within a limited number of moves and under 3 lives! </p>
